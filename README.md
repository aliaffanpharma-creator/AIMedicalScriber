# AI Medical Scriber for Pakistan

- Record doctor-patient conversation
- AI generates SOAP note (Groq Llama 3.3 / Whisper)
- Edit, save locally (IndexedDB), export PDF

## Deploy to Vercel (free)

1. Push this repo to GitHub
2. Import project on Vercel
3. Add environment variable: `GROQ_API_KEY`
4. Deploy

No credit card required. Groq free tier is generous.
